# Atlas Gyms Website

6-Week Transformation Program website for Atlas Gyms in Harrogate & York.

## Features

- Responsive design for all devices
- Hero section with transformation stats
- Program information sections
- Client transformation gallery
- Contact/CTA sections

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Open browser to `http://localhost:3000`

## Deployment

This site is deployed on Vercel at: https://www.atlas-gyms.co.uk

## Assets Needed

- Add transformation photos to `assets/` directory
- Update client testimonials
- Add any additional images or media
