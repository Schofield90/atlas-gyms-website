⚠️ Important: LeadDec Dashboard Configuration

To ensure forms redirect properly to the booking pages, you need to configure the redirect URLs in your LeadDec dashboard:

  1. Log into your LeadDec dashboard
  2. Navigate to Forms → Form Settings
  3. For each form, set the "Redirect URL after submission" to:
    • Harrogate form: https://atlas-gyms-website.vercel.app/harrogate-booking.html
    • York form: https://atlas-gyms-website.vercel.app/york-booking.html
  4. Save your changes

JavaScript Form Handling

The website includes JavaScript to intercept form submissions and control redirects. This ensures users stay on your site after submitting the form.

How it works:

  1. Monitors the form iframe for submission events
  2. Intercepts any redirect attempts
  3. Redirects to the appropriate booking page
  4. Prevents the form from taking over the entire browser window
// Example of the form handling code
window.addEventListener('message', function(event) {
    if (event.data && event.data.type === 'form-submitted') {
        window.location.href = '/booking-page.html';
    }
});

Testing the Forms

To test the form submission flow:

  1. Visit either /harrogate.html or /york.html
  2. Fill out the consultation form
  3. Submit the form
  4. Verify you're redirected to the appropriate booking page
  5. Check that the booking widget loads correctly

Troubleshooting

If forms are not redirecting properly:

  • Check browser console for JavaScript errors
  • Verify form IDs match in the HTML
  • Ensure LeadDec dashboard has correct redirect URLs
  • Clear browser cache and try again
  • Check that booking pages exist at the correct paths
